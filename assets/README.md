# Assets Folder

Add your custom images and GIFs here:

## Recommended Assets

### Banner (Top of README)
- `banner.gif` - Shadow in the Eminence themed banner (1200x400px)
- `banner.png` - Static alternative

### Dividers
- `divider.gif` - Animated divider between sections
- `divider.png` - Static divider

### Character Images
- `shadow.gif` - Shadow anime character GIF
- `shadow2.gif` - Alternative Shadow image
- `alpha.gif` - Alpha/Atomic themed image

## Where to Get Assets

1. **Shadow in the Eminence GIFs:**
   - Giphy.com - Search "Shadow Garden"
   - Tenor.com - Search "The Eminence in Shadow"
   - Pinterest - Anime GIFs

2. **Create Your Own:**
   - Use Canva for banners
   - Use EZGIF.com for resizing
   - Recommended: 1200x400px for banners

3. **Free Anime Assets:**
   - Wallpaperflare.com
   - AlphaCoders.com
   - Pinterest (download via savefrom.net)
